# Bangers Font

Download the Bangers font from Google Fonts and place `Bangers-Regular.ttf` in this directory.

**Download link:** https://fonts.google.com/specimen/Bangers

## Manual Installation

1. Go to https://fonts.google.com/specimen/Bangers
2. Click "Download family"
3. Extract the ZIP
4. Copy `Bangers-Regular.ttf` to this directory

The font will be injected into web pages via `web_accessible_resources` in the manifest.
